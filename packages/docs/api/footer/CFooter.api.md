### CFooter

```jsx
import { CFooter } from '@coreui/vue'
// or
import CFooter from '@coreui/vue/src/components/footer/CFooter'
```

#### Props

| Prop name    | Description                           | Type   | Values                | Default |
| ------------ | ------------------------------------- | ------ | --------------------- | ------- |
| **position** | Place footer in non-static positions. | string | `'fixed'`, `'sticky'` | -       |
